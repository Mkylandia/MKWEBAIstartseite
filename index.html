Zuverlässige Prozessunterstützung

ZEISS MCS 700 Spektrometer sind das ideale System, um Ihren Beschichtungsprozess in Echtzeit zuverlässig zu unterstützen. Sie verfügen über ein durchgehendes Bussystem bis zum Messkopf, was den Integrationsaufwand erheblich reduziert.

Vielfältige Schnittstellen

Wir bieten eine breite Palette an Schnittstellen, die Ihnen eine Komplettlösung bieten, die für verschiedene Anwendungen konfiguriert werden kann. Die MCS 700 Familie ermöglicht die Verbindung zusätzlicher Spektrometer oder Lampenmodule.

Anpassbare Lösungen

Die Wahl liegt bei Ihnen: ZEISS MCS 700 Spektrometer können entweder mit ThinProcess® Software verwendet oder für Ihr eigenes System zusammen mit passenden Messköpfen über das ZEISS OSIS SDK maßgeschneidert werden.

Zubehör für MCS 700

Erweitern Sie die Möglichkeiten Ihres Systems

OFR A10c

Mit dem OFR A10c Messkopf ist es möglich, absolute Transmissions- und Reflexionsmessungen durchzuführen. Es sind keine Referenzmaterialien mit bekannten Eigenschaften erforderlich.

OFR 155

Der OFR155 Winkelfarbmesskopf ermöglicht Inline-Messungen und wird zur Farbauswertung und spektralen Reflexionsmessung mit einem Beobachtungswinkel von 55° verwendet.

OFR 104

Der OFR 104 Standard-Farbmesskopf ermöglicht Inline-Messungen und wird zur Farbauswertung und spektralen Reflexionsmessung mit einem Beobachtungswinkel von 4° verwendet.

OFR H2c

OFR H2c ist eine Beleuchtungskugel mit interner redundanter Lichtquelle für spektrale Transmissionsmessung von spiegelnden Proben.
